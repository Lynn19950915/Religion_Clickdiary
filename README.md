### 專案簡介

宗教點日記與高中生點日記相似，主要分析用戶的宗教接觸與活動。

差別是個人宗教生活圈大小不一，需要自行建立「接觸清單」，

網站在<b>流程簡化、直覺操作、動態回饋</b>等方面做了 UI/UX 的優化。
